3. Algorithm Logic (DP)

dp = array of size (amount + 1) filled with ∞
dp[0] = 0
for i from 1 to amount:
for each coin in sorted_coins:
if coin <= i:
remainder = i - coin
potential = dp[remainder] + 1
if potential < dp[i]:
dp[i] = potential
else: break

Key Concepts

Dynamic Programming solves complex problems by breaking them down into simpler subproblems. It stores the results of these subproblems in a table (memoization) to avoid redundant calculations, ensuring an optimal solution.

CLO-2: Role of Sorting

Before running DP, we sort the coins ascending. If we encounter a coin larger than the current amount i, we know all subsequent coins will also be too large. We safely break the loop early.
